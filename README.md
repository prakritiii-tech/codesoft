# CODSOFT Data Science Internship

This repository contains the Machine Learning projects completed during the CodSoft Data Science Internship.

## Projects Included

### 1. Titanic Survival Prediction
Predicting passenger survival using Machine Learning algorithms.

### 2. Iris Flower Classification
Classifying iris flowers into different species using classification models.

### 3. Sales Prediction Using Python
Predicting product sales based on advertising expenditure.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithms

- Logistic Regression
- Random Forest
- Decision Tree
- Linear Regression
