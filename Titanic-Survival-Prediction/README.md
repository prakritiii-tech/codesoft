# Titanic Survival Prediction

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques.

## Dataset
Titanic Dataset containing passenger details such as:
- Passenger Class
- Gender
- Age
- Fare
- Family Information
- Embarkation Port

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction

## Results
- Logistic Regression Accuracy: ~80%
- Random Forest Accuracy: ~83%

## Key Insights
- Female passengers had higher survival rates.
- First-class passengers survived more frequently.
- Fare and family size influenced survival probability.
