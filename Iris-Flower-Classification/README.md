# Iris Flower Classification

## Project Overview
This project classifies iris flowers into three species using Machine Learning algorithms.

## Species
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Models
- Decision Tree Classifier
- Random Forest Classifier

## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Visualization
4. Model Training
5. Prediction
6. Performance Evaluation

## Results
The model successfully classified iris flowers with high accuracy.

## Key Insights
- Petal length and petal width were the most important features.
- Random Forest performed better than Decision Tree.
